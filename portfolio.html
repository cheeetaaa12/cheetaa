Organisasi Keagamaan Sekolah

Rohani Islam (ROHIS)

Aktif sebagai anggota Rohani Islam di sekolah. Berpartisipasi dalam berbagai kegiatan keagamaan seperti kajian rutin, peringatan hari besar Islam, dan kegiatan sosial berbasis nilai-nilai islami untuk mempererat ukhuwah di lingkungan sekolah.

Organisasi Kepemimpinan Sekolah

OSIS (Organisasi Siswa Intra Sekolah)

Berperan aktif dalam OSIS sebagai bagian dari penggerak kegiatan sekolah. Terlibat dalam perencanaan dan pelaksanaan berbagai event sekolah, pengembangan program siswa, serta melatih jiwa kepemimpinan dan kerja sama tim secara nyata.

Project Game Dev Kolaborasi

Project Alice — Membuat Game

Proyek kolaborasi pembuatan game bernama Alice bersama tim. Mengerjakan perancangan alur permainan, karakter, dan mekanisme gameplay. Proyek ini mengasah kemampuan berpikir logis, kerja sama tim, dan kreativitas dalam dunia pengembangan game.
